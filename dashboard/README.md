# Dashboard Guide

Build the Power BI and Tableau dashboards from the processed CSV outputs generated by the notebook:

- `data/processed/historical_sales.csv`
- `data/processed/store_metrics.csv`
- `data/processed/store_forecasts.csv`

## Recommended Dashboard Sections

### 1. Executive Summary
KPIs:
- Total historical sales
- Average forecasted weekly sales
- Average MAPE
- Best forecasted store
- Hardest-to-forecast store

### 2. Store Forecast Explorer
Visuals:
- Store filter/slicer
- Historical vs forecasted weekly sales line chart
- Store-level MAE, RMSE, and MAPE cards

### 3. Model Performance
Visuals:
- MAPE by store
- RMSE by store
- Top 10 best forecasted stores
- Top 10 hardest-to-forecast stores

### 4. Seasonality and Business Drivers
Visuals:
- Average sales by month
- Holiday vs non-holiday weekly sales
- Top and bottom stores by historical sales

## Power BI Notes
Create relationships using `Store` as the main key. Use a dedicated Date table if you want more robust time intelligence.

## Tableau Notes
Use relationships or joins based on `Store`; use Date fields for time series visuals. Add Store as a dashboard filter and apply it to all relevant sheets.
